l# 🏦 End-to-End Financial Risk & Credit Assessment Analysis

## 📌 Project Overview
This project provides a comprehensive analysis of over 8,000 loan records to identify and mitigate financial risks. By integrating data from **Excel, SQL, Power BI, and Tableau**, I have built a multi-tool dashboard system that helps financial institutions make data-driven lending decisions.

## 📊 Key Analytical Insights
- **Portfolio Health:** Monitored a total loan disbursement of **$215.4M** with an average credit score of **699.08**.
- **Risk Identification:** Highlighted a critical **67.79% Default Ratio**, necessitating immediate policy interventions.
- **Demographic Analysis:** Identified the **20-40 age group** as the highest volume segment, requiring dynamic credit limit scaling.
- **Geographical Risk:** Visualized global loan distribution using Tableau Maps to identify high-concentration risk zones.

## 💡 Strategic Recommendations
1. **Dynamic Credit Limit Scaling:** For the 20-40 age group, implement credit limits that increase only after a proven 24-month clean repayment history.
2. **Income-Based Capping:** Limit Debt-to-Income (DTI) ratios to 35-40% for mid-income segments to prevent immediate defaults.
3. **Asset-Backed Incentives:** Offer lower interest rates to borrowers who provide higher asset collateral.

## 🛠️ Tools & Technologies
- **Excel:** Pivot Tables, Slicers, and DTI Analysis.
- **Power BI:** Premium Dark-Theme dashboards for Executive Reporting.
- **Tableau:** Global Risk Mapping and Demographic Binning.
- **SQL:** Data cleaning and schema optimization (BIGINT/DECIMAL handling).
